# QuickDock

Windows 桌面快捷启动器，可吸附在屏幕左右边缘，快速打开文件、文件夹和应用程序。

## 功能特性

- **快捷启动**：支持文件、文件夹、可执行程序、快捷方式（`.lnk`）
- **分类管理**：多分类组织，支持新建、重命名、删除
- **拖拽添加**：从资源管理器拖入文件/文件夹，可插入到指定位置
- **拖拽排序**：拖动已有图标调整顺序
- **拖出到外部**：将项目拖到桌面、文件夹或其他应用窗口
- **边缘吸附**：窗口可吸附到屏幕左/右侧，收起后显示细条指示
- **靠近呼出**：鼠标靠近边缘自动展开（可配置灵敏度）
- **双击中键呼出**：可选模式，需双击中键才展开；拖文件到收起细条也会展开
- **系统托盘**：隐藏到托盘，右键菜单显示窗口或退出
- **开机自启**：可在设置中开启
- **深色主题**：现代化 UI，自定义图标与名称

## 环境要求

- Windows 10 / 11
- [.NET 10 SDK](https://dotnet.microsoft.com/download)

## 快速开始

```bash
# 克隆后进入项目目录
cd WpfApp1

# 编译
dotnet build WpfApp1/WpfApp1.csproj

# 运行
dotnet run --project WpfApp1/WpfApp1.csproj
```

编译产物为 `QuickDock.exe`，位于 `WpfApp1/bin/Debug/net10.0-windows/`（或 Release 目录）。

## 使用说明

### 添加项目

| 方式 | 操作 |
|------|------|
| 按钮添加 | 点击标题栏 **＋**，选择文件或文件夹 |
| 拖拽添加 | 将文件/文件夹拖入右侧列表区域 |
| 边缘拖入 | 收起状态下，将文件拖到屏幕边缘细条上自动展开 |

### 打开与拖出

- **双击**图标：打开项目（工作目录为文件所在文件夹）
- **拖动**图标：拖到其他窗口（如资源管理器、聊天工具附件区）

### 分类

- 左键选择分类
- **右键**分类：重命名 / 删除

### 项目菜单

**右键**图标：

- 修改名称和图标
- 删除

### 窗口操作

| 操作 | 说明 |
|------|------|
| 拖动标题栏 | 移动窗口（自由模式下） |
| 拖到屏幕边缘 | 自动吸附左/右侧 |
| 拖离边缘 | 恢复自由位置 |
| **−** 按钮 | 隐藏到系统托盘 |
| **⚙** 按钮 | 打开设置 |

### 设置项

- **开机自启动**：写入当前用户注册表（可能需要 UAC 确认）
- **边框吸附**：启用后窗口可折叠到屏幕边缘
- **吸附位置**：左侧 / 右侧 / 自由位置
- **呼出方式**：
  - *靠近呼出*：鼠标靠近边缘即展开
  - *靠近 + 双击中键呼出*：需双击中键展开；拖文件到收起细条也会展开
- **靠近灵敏度**：3–30 像素，默认 20

## 数据存储

配置与项目数据保存在：

```
%AppData%\QuickDock\data.json
```

## 项目结构

```

├── WpfApp1/

│   └── Assets/        # 应用图标
└── README.md
```

## 开发说明

### 重新生成应用图标

```bash
dotnet run --project WpfApp1/WpfApp1.csproj -- --generate-icon
```

图标会写入 `WpfApp1/Assets/AppIcon.ico`。

### 构建 Release

```bash
dotnet build WpfApp1/WpfApp1.csproj -c Release
```
